ระบบนี้ใช้วิธีการเผยแพร่ Google Sheets เป็นสาธารณะ (Public) ไม่ต้องใช้ API Key

1 เปิด Google Sheets ของคุณ

เปิด Sheets ที่ต้องการแสดงข้อมูล: คลิกที่นี่

2 เผยแพร่เป็นสาธารณะ

ไปที่ File → Share → Publish to web → เลือก "Entire document" → คลิก Publish

คลิกปุ่มด้านล่างเพื่อเริ่มใช้งานระบบ
